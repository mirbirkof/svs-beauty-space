API-документація

Інтерактивна специфікація публічного API салону (/api/v1) для інтеграцій: сайт, мобільний застосунок, партнери.
Авторизація — за API-ключем (заголовок x-api-key або Authorization: Bearer <key>). Ключі створюються у розділі «Система → API-ключі». Натисніть Authorize у формі нижче, щоб тестувати запити прямо звідси.
